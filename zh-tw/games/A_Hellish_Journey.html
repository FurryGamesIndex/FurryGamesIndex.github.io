地獄之旅 / A Hellish Journey

Microsoft Windows
Apple macOS
GNU/Linux
Android
遊戲故事是關於Aki（名字可自定義），一個為了得到報酬而不在乎工作好壞，能做任何工作的僱傭兵狼。他在一個不知名的國度長大，在被父母遺棄後，變成了一隻流浪狼。直到有一天，他遇到了一個有一顆金子般的心的人，幫助他重新站起來，但這個陌生人有一天要離開他去執行一項重要的任務……

然後，他又變回了孤身一人，遵循著陌生人教給他的東西，把這些技能當作成為僱傭兵的籌碼。但他的行為方式太過了，很快就出格了，因為一項工作出了大錯而激怒了國王。在被要求辭去僱傭兵工作或被流放的選擇下，他選擇了後者，導致他最終被王國的騎士追殺。

但在他完全死去之前，突然發生了一件意想不到的事情，一個臭名昭著的惡魔對他產生了興趣，並與他達成契約。Aki成為了他的容器，獲得了難以想象的力量和永生的生命。

有了新獲得的力量，Aki和惡魔使用了一個完全出乎意料的咒語，這吸引了其他一些惡魔的注意。他們是誰？他們是來幫助他的嗎？您能和他們其中一個發展關係嗎？又或者其中一個是來取Aki的性命的？……故事即將開幕。
