わんダフルとらブル if

バイト先の居酒屋は今日も大盛況！

お客も帰ってようやく片付け…と、客間の扉を開けるとそこには想像もしていない光景が…！

漫画版「わんダフルとらブル シリーズ（1-4）」、小説版「わんダフルとらブル！」のシナリオを元に書き下ろされた、「if」の世界線のストーリー！

バンダナ大学生 白岳 和、虎オッサン 霧島 漸次、それに熊オッサン 二階堂 吉右衛門の三人が織りなす、ちょっとエッチでドキドキハラハラなトラブルをお楽しみください！

詳細は以下のリンクからどうぞ！

  • 公式Twitter： https://twitter.com/orekemo
  • サークル公式HP： http://www.cilcafe.com/
  • オープニングムービー：https://youtu.be/51c-rec03vg

対応デバイス

対応OS : Windows10 のみ動作確認済、MacOS 等の他 OS では動作しません。

ゲームの詳細

選択肢によって、エンディングが変わるマルチエンドの短編アドベンチャー（ビジュアルノベル）ゲームです。

全編掻き下ろしによるイラストをお楽しみください！
