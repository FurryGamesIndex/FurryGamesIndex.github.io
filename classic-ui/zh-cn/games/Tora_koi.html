トラ☆コイ！

『トラ☆コイ！』無料体験版です。

虎獣人のおっさん、おっちゃん、おじさんとの恋愛ＡＤＶゲームです。

※同性愛表現および性的描写が含まれます。

起動方法

「torakoi.exe」を実行してください。

※本ゲームをプレイする際は、「torakoi_taiken」ファイルを右クリックし、対象ファイルを展開してから実行して下さい。

解凍しない状態(zipファイルのまま)で実行すると、正常にプレイできない可能性があります。

推奨環境

OS: Windows Vista/7/8/10 (64bit)
CPU : Pentium III 800MHz以上 ( 1GHz以上推奨 )
メモリ: 128MB以上 ( 1GB以上推奨 )
HDD: 空き容量2GB以上
解像度: 1280x720以上
※「吉里吉里Z／KAG3」の動作条件に準じます。

ただし、これらの条件を満たす場合でも、100％の動作を保証するものではありません。
