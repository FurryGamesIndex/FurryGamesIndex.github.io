阿達斯特拉 / Adastra

Adastra 是一部混合有大量科幻與政治陰謀的戀愛視覺小說。你將面臨帝國的動盪險境，你需要決定哪些人可以相信，而哪些人絕對不能。

拱橋 / Arches

《拱橋》是一個獨立故事，講述的是迴音鎮事件發生以後的故事。
一對伴侶來到迴音鎮做一些超自然事件的調查，這座小鎮自從集體癔症爆發後已經廢棄了五年。儘管如此，一些邪惡的東西仍然潛伏在這座小鎮之中，足以使恐懼延續下去。

迴音 / Echo

"我就知道你會回來的... 即使再次回到這裡，你也會試圖逃離... 停下吧... 因為就像這個鎮子一樣... 你永遠都在原地打轉..."

回聲：65號公路 / Echo: Route 65

《回聲：65號公路》是一部講述以西南鄉鎮少年 Chase 為中心故事的視覺小說。Chase 以非常尷尬的形式被父母發現是同性戀。從放學回家的公交車上下車後，他為了逃避自己“出櫃”的事實，沿著一條稱為 65 號線的公路進行了一場夜間冒險。 《迴音》系列有三部主要作品，以及一部短篇前傳。按時間順序來講，分別是：設定在1915年的前傳作品《吸菸室》，主角團在2008年萬聖節探險的短篇前傳《迴音：65號公路》，系列首部作品《迴音》，以及《迴音》事件發生5年後的續作《拱橋》。

阿達斯特拉：幕間 / Adastra:Interea

Adastra: Interea (也可簡稱為Interea) 是阿達斯特拉系列的劇情擴充包。作者稱，Interea將會是一部短篇小說。

克希米亞 / Khemia

克希米亞是一部獸人科幻視覺小說，為阿達斯特拉的正統續作。本作將以全新主角Scipio的視角，從阿達斯特拉出發，開啟探索克希米亞的旅途。

聖誕特輯 / Smoke Room Christmas

這是2021年聖誕節期間，Echo Project團隊於Patreon上公開放出的The Smoke Room的現代版本。據編輯George的解釋，遊戲發生背景為平行宇宙(AU)，劇情與原版無關聯。

吸菸室 / The Smoke Room

本作故事發生於一個世紀前的迴音鎮，遊戲視角從一名性工作者 Samuel 的身上展開，他那努力改善生活的願景，卻為自己帶來了殺身之禍。
