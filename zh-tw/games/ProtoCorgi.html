星際奇基

Microsoft Windows
Apple macOS
GNU/Linux

《星際奇基》是一款復古像素風格的水平射擊遊戲，包含電子遊戲機玩法，讓玩家操控 C3 級 (可愛機械科基) 小狗 Bullet 。

準備好在這款可愛的狗狗射擊遊戲中射擊、閃躲並發出汪汪戰吼吧！數不清的敵人一波波來襲，包括巨大的首領和各種致命危險的障礙，考驗你的技巧。幫助 Bullet 完成這趟旅程，從可怕的外星人手中救出主人和好友妮希！

別擔心，人人都可藉由調整難度來進行這場冒險，難度從「小狗」級到「超絕惡夢」級任君選擇。

關門，放狗

最多可帶上 27 種不同武器組合來奮勇退敵！身為真正的 C3 級機械狗， Bullet 使用的絕不是傳統武器。利用 3 種不同大規模攻擊來掃除敵軍，或施展你的戰鬥魂暴打敵人！

刺激競技

善用汪汪力量，憑藉技巧登上風雲榜！願你留名青史，無可匹敵。讓朋友知道誰才是最厲害的柯基！

創作與分享

充分揮灑創造力！《星際奇基》提供平台編輯模式，你可以在這裡盡情創造出你喜歡的各種關卡，並分享給朋友一同遊玩。

汪汪隊大集結！

世間萬狗，叫聲不同，請盡情自訂 Bullet 的叫聲吧！
