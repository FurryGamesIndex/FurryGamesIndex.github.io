阿達斯特拉 / Adastra

一切都要從你在羅馬的留學計劃開始說起。你才剛剛安頓下來，就突然被一個外星人所綁架，他想從你這裡得到什麼？嗯，看來他沒有告訴你的打算。在反應過來之前，你已經離地球數百萬英里遠了，等待你的是一個你一無所知的帝國。
在背叛和暗算的渦流中，你漸漸懷疑自己的綁架者也許才是最能夠信任的。

Arches

《Arches》是一個獨立故事，講述的是迴音鎮事件發生以後的故事。
一對伴侶來到迴音鎮做一些超自然事件的調查，這座小鎮自從集體癔症爆發後已經廢棄了五年。儘管如此，一些邪惡的東西仍然潛伏在這座小鎮之中，足以使恐懼延續下去。

回聲 / Echo

在春假中，Chase，一個出生在迴音鎮的年輕水獺，因學校專案回到小鎮，想要記錄1870年代所發生的造成了該鎮如今荒涼現狀的事件。同他一起在迴音鎮長大的5個朋友，也藉此機會能重聚一堂。不幸的是，Chase逐漸意識到他們在迴音鎮的回憶似乎遭到了美化。回到小鎮，他們挖掘出過去許多痛苦的回憶；發現了多年來隱藏於表面之下的，能夠使他們關係完全破裂的秘密。而最重要的，Chase漸漸發現迴音鎮也有自己的秘密。那些他以為是因為兒時想象力過剩所造成的古怪而無法解釋的回憶，正真實地發生在他們的眼前。

回聲：65 號線 / Echo: Route 65

《回聲：65 號線》是一部講述以西南鄉鎮少年 Chase 為中心故事的視覺小說。Chase 以非常尷尬的形式被父母發現是同性戀。從放學回家的公交車上下車後，他為了逃避自己“出櫃”的事實，沿著一條稱為 65 號線的公路進行了一場夜間冒險。

阿達斯特拉：幕間 / Adastra:Interea

Adastra: Interea (也可簡稱為Interea) 是Adastra系列的劇情擴充包，為系列作者Howly失去對Khemia的創作熱情後目前唯一仍在更新的作品，
Howly對外宣稱，Interea將會是一部短篇VN。

克希米亞 / Khemia

Khemia / 克希米亞是一部獸人科幻VN，為Adastra的正統續作。本作將以全新主角Scipio的視角，從阿達斯特拉出發，開啟探索克希米亞的旅途。

聖誕特輯 / Smoke Room Christmas

這是2021年聖誕節期間，Echo Project團隊於Patreon上公開放出的The Smoke Room的現代版本。據編輯George的解釋，遊戲發生背景為平行宇宙(AU)，劇情與原版無關聯。

煙室 / The Smoke Room

《煙室》是一部以1915年的沙漠小鎮為背景的恐怖/愛情視覺小說，為《回聲》的獨立前傳。 搬到 Echo 鎮真是個壞主意，輕信初次相識的人讓一切更加糟糕。《煙室》講述了 Samuel 的故事，他是一個性工作者，在鎮上最臭名昭著的妓院工作。在他犯了自己人生中最大的錯誤後，Sam 想要竭力掩蓋秘密。然而，他很快意識到這麼做幾乎是不可能的… Echo 鎮似乎知道他所做的一切，絕不讓他輕易遺忘。
